## ğŸ” Phase 2: Firebase Integration

You will use Firebase for two things: **Authentication** (Logins) and **Firestore** (Database to save user history).

1. **Google Login:** Enable "Google" in the Firebase Auth console.
2. **Firestore:** Create a collection called `analyses` to store resume results.
3. **Security Rules:** Set rules so users can only read/write their own data (`request.auth.uid == resource.data.userId`).

## ğŸ§  Phase 3: AI Integration (The Mentor)

Since this is a frontend-heavy project, you have two choices for the AI:

* **Direct API Call (Quickest):** Call the Gemini/OpenAI API directly from your `app.js`.
* **Vercel Serverless Function (Most Secure):** Place the API call in a small script in an `/api` folder. This prevents your API key from being visible in the browser's "Inspect Element."

> **Warning:** Never hardcode your API key in `index.html`. Always use Vercel's **Environment Variables** dashboard.
